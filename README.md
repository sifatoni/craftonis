# Craftonis
**HR Intelligence Platform** — Hire Smart. Manage Better. Grow Faster.

## Modules
1. Dashboard & Analytics
2. Lead Generation (Token-based)
3. Job Posting & Candidate Pipeline
4. CV & Scoring Engine
5. Technical & Functional Interview Suite
6. Meeting Ledger (WebRTC Video Call)
7. Meeting Minutes Generator
8. AI-Powered Onboarding
9. HRM Suite
10. Workspace & Billing Admin

## Tech Stack
- **Web**: Next.js 14, TypeScript, TailwindCSS
- **API**: NestJS, TypeScript, PostgreSQL, Prisma, Redis
- **Mobile**: React Native (Expo)

## Getting Started
```bash
npm install
npm run dev
```
